# Employee Pension Calculator – Manual Testing Project

## Project Overview
This project demonstrates manual testing of a simple Employee Pension Calculator web application.  
The application calculates the estimated monthly pension based on employee salary and years of service.

## Application
The calculator is developed using HTML and JavaScript.

Pension Calculation Formula:

Pension = Salary × Years of Service × 0.015

## Project Files

pension_calculator.html  
Simple web application used for pension calculation.
