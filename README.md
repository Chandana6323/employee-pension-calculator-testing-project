# Employee Pension Calculator – Manual Testing Project

## Project Overview
This project demonstrates manual testing for a simple Employee Pension Calculator web application.

## Application
HTML-based calculator that calculates pension using:

Pension = Salary × Years of Service × 0.015

## Testing Documents
The testing documentation includes:

• Test Scenarios
• Test Cases
• Test Execution Report
• Defect Report

All documents are organized in an Excel workbook with separate sheets.

## Technologies Used
HTML
Manual Testing
Excel Documentation
